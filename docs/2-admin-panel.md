# Rullst Admin Panel

Rullst ORM includes a completely free, drop-in web dashboard that automatically reads your `#[derive(Orm)]` models and lets you manage your database graphically. 

Unlike traditional ORMs that charge for their studio/admin tools, Rullst provides this natively.

## How it Works

The Admin Panel is simply a gorgeous HTML/CSS/JS frontend generated by the ORM itself. It relies on standard REST endpoints that you expose via your favorite web framework.

### Example using Axum

To serve the dashboard, you just map the `dashboard_html()` function to a route:

```rust
use axum::{routing::get, Router, response::Html};
use rullst_orm::dashboard_html;

#[tokio::main]
async fn main() {
    let app = Router::new()
        .route("/admin", get(|| async { Html(dashboard_html()) }));

    let listener = tokio::net::TcpListener::bind("127.0.0.1:3000").await.unwrap();
    println!("Admin Panel available at http://localhost:3000/admin");
    
    axum::serve(listener, app).await.unwrap();
}
```

## Features

- **Dark Mode Native**: A stunning, premium dark interface out-of-the-box.
- **Audit Logs View**: Directly inspect `rullst_audits` to see revision history.
- **Zero Config**: No React/Vue compilation required. Just serve the string!
