# The Most Productive Rust ORM

Rullst ORM is a Laravel-inspired, database-agnostic Object Relational Mapper for Rust.

## Why Rullst ORM?

- **Zero Boilerplate:** Use `#[derive(Orm)]` and immediately gain access to a powerful Query Builder.
- **Full-Text Search:** Native integration with Algolia, Meilisearch, and database `LIKE` fallbacks via the `#[orm(searchable)]` macro.
- **Rullst Admin Panel:** Instantly deploy a beautiful dark-mode web dashboard to manage your tables natively.
- **Multi-Tenancy:** Effortlessly separate your users' data using magical tenant isolation contexts.

Get started instantly and build the future of Rust backend systems.
